## Installation

1. Clone the repo

   ```sh
   git clone
    ```

2. Install NPM packages

    ```sh
    npm install
    ```

3. Create .env file (copy the .env.example) and enter backend url (check backend repo [here](https://github.com/Murf-y/smoll_backend))

4. Start the server

    ```sh
    npm run dev
    ```

5. Open the app in your browser

    ```sh
    http://localhost:5173/
    ```
